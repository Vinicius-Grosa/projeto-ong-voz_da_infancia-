# projeto-ong-voz_da_infancia-
Sistema de Gestão de Bazar Solidário

 📌 Descrição
Este projeto tem como objetivo desenvolver um sistema para gerenciamento de um bazar beneficente, permitindo controle de vendas, doações e relatórios.

🎯 Objetivo
Facilitar a organização e administração do bazar, melhorando o controle de produtos e arrecadações.

🛠️ Ferramentas utilizadas
- Figma (protótipos)
- GitHub (versionamento)
- Trello (gestão de tarefas)
